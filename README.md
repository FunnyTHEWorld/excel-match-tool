# 运行和部署您的 AI Studio 应用

这里包含了在本地运行您的应用所需的一切。

在 AI Studio 中查看您的应用：https://ai.studio/apps/drive/1DlBoBaUz1xR4dtavjsOn1z50WepFjKQY?showPreview=true

## 本地运行

**先决条件：** Node.js


1. 安装依赖：
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. 运行应用：
   `npm run dev`
